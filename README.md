# Finance Backend - Zorvyn Assignment

This is a professional backend setup for a Financial Management System built with **Node.js**, **Express**, and **MongoDB**.

## 🚀 Features
- **Requirement 1:** Clean Folder Structure (Models, Routes, Middleware).
- **Requirement 2:** CRUD operations for Financial Records (Income/Expense).
- **Requirement 3:** Dashboard Summary API (Calculates Total Income, Expense, and Balance).
- **Requirement 4:** Role-based Access Control (Admin/Analyst/Viewer).

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Security:** JWT & Role-based Middleware

## 📁 Folder Structure
- `models/`: Database schemas for Users and Records.
- `routes/`: API endpoints for transactions and dashboard.
- `app.js`: Main entry point of the application.

## 🏃 How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server using `node app.js`.
